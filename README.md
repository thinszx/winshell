# 操作系统课程设计

40. 操作系统接口：兼容Unix/Linux 命令接口3。（1-2 人，难度：5）
- 为Windows 操作系统建立一个兼容Unix/Linux 的shell 文件执行命令接口；
- 核心是实现.sh 格式文件的解读和执行
- 应该包括常用的shell 文件的流程控制语句，如顺序、循环和分支转移等，命令的内容与详细格式请查阅Unix/Linux 命令手册；
## 已完成功能

- [x] 转义字符 `\` 
- [x] 通过 `;` 换行
- [ ] 注释的识别与跳过
- [ ] 通过 `ctrl+c` 结束语句执行或循环，但不结束shell

- [ ] 添加 `ls` `cd` `pwd` 等内置命令
### if分支语句
